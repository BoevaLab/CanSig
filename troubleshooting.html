Troubleshooting#

  1. GSEA will not run

    • Is your gene annotation consistent? If your genes are annotated using Entrez in your original expression object, then you should run GSEA using an adapted .gmt file. You can download files for official gene ID and for Entrez on the MSigDB website (https://www.gsea-msigdb.org/gsea/downloads.jsp#msigdb). If your genes are annotated using another system, you should first translate into Official Gene ID or Entrez.
